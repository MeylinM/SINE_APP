MENÚ PRINCIPAL

Bienvenido/a al manual de uso de la aplicación SINE Ingeniería Eléctrica. Esta aplicación está dirigida a los empleados para que puedan llevar la gestión de las bobinas.

Selecciona una sección para obtener ayuda:

  • Escanear QR de las Bobinas
  • Introducir Bobinas manualmente
  • Mostrar todos los datos de las bobinas
  • Mostrar Bobinas a Devolver

ESCANEO DE QR

¿Qué puedes hacer aquí?

  • Escanear un código QR.
  • Introducir información desde cero.
  • Modificar información existente.

¿Cómo escanear el QR?

Después de pulsar "ESCANEAR QR", enfoca el código con la cámara.
Aparecerá un mensaje de confirmación para asegurarte de que es el código correcto
y evitar escaneos accidentales.

Introducir o modificar bobinas

  • Si la bobina ya existe: se cargarán automáticamente los datos guardados previamente. Solo se podrán modificar los campos habilitados según el estado actual:
    • Para Devolver: se podrán editar los campos Información Para Devolver y Observaciones.
    • Devuelto: se podrán editar los campos Información Devuelto y Observaciones.
    • El botón Agregar Observaciones siempre estará disponible, sin importar el estado.
      Al pulsarlo, se abrirá un recuadro donde podrás añadir una nueva observación.
      • El botón Agregar guarda en la lista la observación escrita.
      • El botón Cancelar permite descartar el texto en caso de error o si decides no añadir nada.
  • Si la bobina no existe: se introducirán los datos desde cero, basados en el código QR escaneado. Se podrán rellenar todos los campos correspondientes al estado Recibido, excepto los reservados para estados posteriores.

Botones Guardar, Limpiar y Cancelar

  • El botón Guardar almacena de forma definitiva la información introducida.
  • El botón Limpiar borra todos los campos del formulario, excepto el código QR escaneado.
  • El botón Cancelar descarta los cambios y regresa a la pantalla anterior sin guardar nada.

Introducir manualmente

¿Qué puedes hacer aquí?

  • Buscar por matrícula.
  • Modificar información existente.

¿Cómo Buscar por matrícula?

Introduce la matrícula en su campo y pulsa el botón con forma de lupa.
El sistema comprobará si existe una bobina asociada a esa matrícula.

Modificar bobinas

  • Si la bobina ya existe: se cargarán automáticamente los datos guardados previamente. Solo se podrán modificar los campos habilitados según el estado actual:
    • Para Devolver: se podrán editar los campos Información Para Devolver y Observaciones.
    • Devuelto: se podrán editar los campos Información Devuelto y Observaciones.
    • El botón Agregar Observaciones siempre estará disponible, sin importar el estado.
      Al pulsarlo, se abrirá un recuadro donde podrás añadir una nueva observación.
      • El botón Agregar guarda en la lista la observación escrita.
      • El botón Cancelar permite descartar el texto en caso de error o si decides no añadir nada.
  • Si la bobina no existe: aparecerá un mensaje indicando que no se ha encontrado ninguna bobina con esa matrícula. No será posible añadir una nueva bobina desde esta pantalla.

Botones Guardar, Limpiar y Cancelar

  • El botón Guardar almacena de forma definitiva la información introducida.
  • El botón Limpiar borra todos los campos del formulario, excepto el código QR escaneado.
  • El botón Cancelar descarta los cambios y regresa a la pantalla anterior sin guardar nada.

Mostrar Todos Los Datos

¿Qué puedes hacer aquí?

  • Visualizar todos los datos registrados en una tabla.
  • Filtrar los resultados por matrícula.
  • Filtrar los datos por estado.
  • Filtrar por empleado responsable.

¿Cómo Ver todos los datos?

Al pulsar el botón "Mostrar Todos los Datos", se abrirá una tabla que muestra todas las bobinas registradas en el sistema.

  • Si has realizado algún filtrado, puedes volver a mostrar todos los resultados simplemente borrando el texto del campo de búsqueda. La tabla se actualizará automáticamente en tiempo real.

Filtrar por Matrícula

  • Si la matrícula existe: se mostrarán los datos correspondientes.
  • Si la matrícula no existe: la tabla aparecerá vacía.

Filtrado por Estado

Podrás visualizar únicamente las bobinas en los siguientes estados:

  • Recibido: bobinas recién registradas.
  • Para Devolver: pendientes de devolución.
  • Devuelto: ya devueltas y archivadas.

Filtrar por Empleado

  • Si el empleado existe: se mostrarán sus bobinas.
  • Si no existe: la tabla no mostrará resultados.

Mostrar Bobinas a Devolver

¿Qué puedes hacer aquí?

  • Visualizar los datos en estado Para Devolver.
  • Filtrar por matrícula o empleado responsable.

¿Cómo Ver todos los datos?

Al pulsar "Mostrar Bobinas a Devolver", verás las bobinas en dicho estado.

  • Para quitar filtros, borra el campo de búsqueda.

Filtrar por Matrícula

  • Si existe: se mostrarán las bobinas correspondientes con estado Para Devolver.
  • Si no existe: la tabla aparecerá vacía.

Filtrar por Empleado

  • Si existe: verás todas sus bobinas en estado Para Devolver.
  • Si no existe: no aparecerán resultados.
