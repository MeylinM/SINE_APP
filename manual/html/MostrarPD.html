Mostar Bobinas a Devolver

¿Qué puedes hacer aquí?

  • Visualizar todos los datos registrados en una tabla con el estado Para Devolver.
  • Filtrar los resultados por matrícula.
  • Filtrar por empleado responsable.

¿Cómo Ver todos los datos?

Al pulsar el botón "Mostrar Bobinas a Devolver", se abrirá una tabla que muestra todas las bobinas registradas en el sistema con el estado Para Devolver.

  • Si has realizado algún filtrado, puedes volver a mostrar todos los resultados simplemente borrando el texto del campo de búsqueda. La tabla se actualizará automáticamente en tiempo real, sin necesidad de salir de la pantalla o reiniciar la aplicación.

Filtrar por Matricula

  • Si la matrícula existe: se mostrarán los datos correspondientes a la bobina registrada con ese identificador con el estado Para Devolver.
  • Si la matrícula no existe: la tabla aparecerá vacía, indicando que no hay resultados asociados.

Filtrar por Empleado

  • Si el empleado existe: se mostrarán todas las bobinas asociadas a dicho empleado con el estado Para Devolver.
  • Si el empleado no existe: la tabla no mostrará ningún resultado.
